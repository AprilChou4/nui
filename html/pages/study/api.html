API

引入Nui.js后，Nui会被注册为一个全局对象，下面介绍一下对象的属性和方法：

note:参数末尾包含“?”符号表示该参数非必填。

load

入口模块调用方法

# 参数

参数名 参数类型 参数说明
id String 模块id，后缀.js可以省略
callback Function 模块加在完毕后（包括内部依赖）回调函数

config

配置模块和目录别名，用以简写模块路径

# 参数

参数名 参数类型 参数说明
key String, Object 
Nui.config('paths', {name:path,...})

Nui.config('alias', {name:path,...})

Nui.config({
	paths:{name:path,...},
	alias:{name:path,...}
})
value Object {name:path,...}

# 说明

paths表示路径别名，集合中base用以设置根目录，用绝对地址定义

alias表示模块别名，集合中可以使用大括号“{目录别名}”简写路径

可直接在id中使用大括号“{目录别名}”引用模块

define

定义封装模块

# 参数

参数名 参数类型 参数说明
id String 模块id
deps Array 模块依赖
fectory 任意类型 工厂
